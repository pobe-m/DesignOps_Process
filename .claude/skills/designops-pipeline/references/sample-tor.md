# Terms of Reference: ระบบ Patient Portal — KindMore Clinic

## วัตถุประสงค์
พัฒนา web application สำหรับผู้ป่วยใช้นัดหมาย, ดูผลตรวจ, และสื่อสารกับแพทย์

## กลุ่มเป้าหมาย
- ผู้ป่วยทั่วไป อายุ 25-60 ปี ใช้มือถือเป็นหลัก
- พยาบาลและ staff คลินิก ใช้ desktop

## Features หลัก
1. ระบบนัดหมาย (จำเป็น)
2. ดูผลตรวจและใบสั่งยา (จำเป็น)
3. Chat กับแพทย์ (ต้องการ)
4. ชำระเงินออนไลน์ (optional)

## ข้อจำกัด
- ต้องผ่าน PDPA compliance
- deploy บน AWS
- เสร็จภายใน Q3 2025

## Design Reference
ต้องการ clean, minimal, medical-grade aesthetic. ดูตัวอย่าง: MyChart, Doctorly
